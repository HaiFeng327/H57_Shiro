Welcome to the H57 Shiro Project

Overview

While working with Shiro, I found myself frustrated by what I saw as a lack of practical examples and tutorials. Particularly around integration with Spring. I decided I'd write my own.

Then I got to thinking, it would be fun to do the whole kit and kaboodle!

  • SpringSource Tool Suite
  • Spring MVC 3.1
  • Apache Maven 3.0
  • Apache Shiro 1.2
  • Hibernate Validator 4.2 : JSR-303 Reference Implementation
  • myBatis 2/0/6
  • mySQL

If you are going to implement all of those, wouldn't it be fun to have a reference implementation that demonstrates more than simply how to implement them? Lets go ahead and highlight some of the interactions. So I started this little project in hopes that someone other than myself will profit from it, and by storing it on GitHub, perhaps we will see some collaborative effort at maintaining and building it out.

Regardless, it's going to be fun for me!
